🔮 AI Forecast
Predictive stock intelligence · EOQ + Velocity model
How This WorksRule-based ML Simulation
The forecast engine analyses each product's current stock, threshold level, and consumption velocity to estimate days until stockout. Reorder quantities use the Economic Order Quantity (EOQ) formula: √(2DS/H) where D = annual demand, S = ordering cost, H = holding cost. In production this connects to AWS SageMaker for real ML inference.
